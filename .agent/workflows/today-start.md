# 指示
以下のことを順番にステップバイステップで行ってください。

# 実行順序
1.  **日付の取得**: 今日の日付を特定する（例：2026-01-10）。
2.  **日誌ファイルの作成**:
    *   テンプレート `.\00_システム\Templates\Daily_Note_Template.md` を読み込む。
    *   今日の日付で新しいファイル `.\05_日誌\YYYY-MM-DD.md` を作成する（YYYY-MM-DDは今日の日付）。
    *   ※ファイルが既に存在する場合は、上書き**しない**こと。そのファイルのパスを使用する。
3.  **過去の日誌を検索**:
    *   `.\05_日誌` 内を検索し、**今日より前で最も新しい日付のファイル（直近の過去ログ）**を見つける。
    *   ※昨日だけでなく、3日前や1週間前でも構わない。一番新しい過去のファイルを探すこと。
    *   **※ファイルが見つからない場合（初回実行時など）**: ステップ4〜6（タスク引き継ぎ関連）をスキップし、ステップ7（Inbox整理）以降に進む。

4.  **昨日の日誌を読み込み**:
    *   **※ファイルが見つかった場合のみ実行**: ステップ3で見つけた日誌ファイルの内容を読み込む。

5.  **タスクの引き継ぎ (Rollover)**:
    *   **重要: ユーザーへのメッセージは、必ず「日本語」で記述すること。**
    *   **未完了タスクのプール化**: 昨日の日誌の「今日のタスク」や「Inbox」、その他のタスクセクションから、未完了タスク（`- [ ]`）を抽出し、今日の日誌の**「未分類 (Inbox / Task Pool)」セクション**に転記する。
    *   **「翌日以降のタスク」セクションの引き継ぎ**: 昨日の日誌の「翌日以降にやろうと思っているタスク」セクションから、未完了タスクを抽出し、今日の日誌の**「翌日以降にやろうと思っているタスク」セクション**に転記する。

6.  **フェーズ1: 事務的タスク移行 (Task Architect)**:
    *   プロンプトファイル `.\00_システム\01_Prompts\Workflow_Prompts\Today_Start\01_Task_Architect.md` を読み込む。
    *   このプロンプトの指示に従い、昨日の日誌から今日の日誌へ、タスクとInboxを「正確に機械的に」転記し、今日の日誌ファイルを更新する。

7.  **フェーズ2: Inbox整理と対話 (Inbox Processor)**:
    *   プロンプトファイル `.\00_システム\01_Prompts\Workflow_Prompts\Today_Start\02_Inbox_Processor.md` を読み込む。
    *   **重要**: ここで一度、ユーザーに対して「Inboxの整理を行いますか？」と問いかけ、Inboxの中身を提示し、対話しながら処理を進める。（タスク化、移動、削除など）

8.  **フェーズ3: 戦略的立案 (Strategic Advisor)**:
    *   Inbox整理が完了したら、プロンプトファイル `.\00_システム\01_Prompts\Workflow_Prompts\Today_Start\03_Strategic_Advisor.md` を読み込む。
    *   `Active_Context.md` と現在のタスクリストに基づき、「今日のスタンス」と「Big 3」を生成し、今日の日誌ファイルに書き込む。

9.  **フェーズ3.5: 秘書のブリーフィング (Secretary Briefing)**:
    *   `Active_Context.md`、`01_価値観(Core_Values).md`、および今日の日誌を読み込む。
    *   プロンプトファイル `.\00_システム\01_Prompts\Workflow_Prompts\Today_Start\03b_Secretary_Briefing.md` を読み込む。
    *   プロンプトの指示に従い、コンテキストに基づいた「提案（Proposals）」や「リマインダー」を生成し、今日の日誌ファイルの `## ☕ 秘書のブリーフィング` セクションに記述する。

10. **フェーズ4: 秘書のタスク整理 (Secretary Scheduler)**:
    *   戦略（Big 3）が決まったら、プロンプトファイル `.\00_システム\01_Prompts\Workflow_Prompts\Today_Start\04_Secretary_Scheduler.md` を読み込む。
    *   プロンプトの指示に従い、「未分類 (Inbox / Task Pool)」にあるタスクを、**「今日のタスク (Today's Tasks)」** セクションにまとめ、整理する。

11. **ユーザーへの通知**:
    *   **重要: 通知メッセージは必ず日本語で記述すること。**
    *   日次セットアップがすべて完了したことを通知し、今日一日を開始するよう促す。
    *   今日の重要タスク（Big 3）を強調して伝える。
