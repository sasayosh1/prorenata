# 指示
以下のステップを順番に実行してください。

# 実行順序
1.  **プロンプトの読み込み**:
    - `00_システム/Prompts/Workflow_Prompts/Daily_Radar/01_Intel_Radar.md` を読み込む。
2.  **情報収集（サーチ）**:
    - 以下のキーワードで `search_web` または `deep-research` を実行。
        - 「AITuber 自律化 最新 2026」
        - 「精神科 看護 AI 効率化 最新」
        - 「デジタルツイン キャラクター 性格再現」
    - 昨日の「AITuberが作れるようになった」などの具体的な文脈があればそれも考慮する。
3.  **分析と要約**:
    - 読み込んだプロンプトの「Output Format」に従い、収集した情報を分析・要約する。
4.  **日誌への追記**:
    - 今日の日付の日誌ファイル（`05_日誌/YYYY-MM-DD.md`）を特定。
    - ファイル内の `## 🧠 思考と感情のログ` の直前に、`## 💡 今日のインテリジェンス` セクションがない場合は作成し、そこに生成した内容を追記する。
5.  **完了通知**:
    - 「インテリジェンス・レーダーが情報をキャッチしました」と報告。
