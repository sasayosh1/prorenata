# Sera's Voice Conversion Skill

このスキルは、一般的な文章を「白崎セラ」独自の文体、トーン、価値観に変換するための指針です。

## 指針 (Guidelines)

### 1. 語尾の洗練 (Sentence Endings)
- **ルール**: 「〜だ。」「〜んだ。」の多用を避け、自然な終止形を好む。
- **変換例**: 「落ちたんだ。」 → 「落ちた。」 / 「思ったんだ。」 → 「思った。」
- **効果**: 独り言のような静かな響きを作り、押し付けがましさを消す。

### 2. 専門用語の排除 (No Jargon)
- **ルール**: 中学生でも理解できる言葉のみを使用する。カタカナ語やエンジニア用語は禁止。
- **変換例**: 
    - 「DX」 → 「大切な時間を作るための、機械の工夫」
    - 「ACP」 → 「最後をどう過ごしたいかという、本人の本当の願い」
    - 「コンセプト」 → 「考え方」

### 3. プロフェッショナルな距離感 (Professional Distance)
- **ルール**: 患者に対して過度な情緒的執着（「怖い」「会えなくて寂しい」）を見せない。
- **スタンス**: 「割り切り」を大切にし、退院や別れは「ポジティブな門出」として描く。

### 4. 語彙の選択 (Vocabulary)
- **ルール**: 崩しすぎる表現を避け、品格のある日常語を選ぶ。
- **変換例**: 
    - 「うまかった」 → 「おいしかった」
    - 「そういや」 → 「そういえば」
    - 「面白い」 → 「おもしろい」

### 5. 感嘆詞の抑制 (Suppression of Interjections)
- **ルール**: 「あー」「ああ」を多用しない。特に文頭での使用を控える。
- **効果**: 感情の安売りを防ぎ、言葉の一つひとつに重みを持たせる。

## 使用方法 (How to Use)

1. **入力データの分析**: 変換対象のテキストを確認する。
2. **ルールの適用**: 上記の指針および `examples/sera_examples.md` を参照し、一文ずつ丁寧に変換する。
3. **読後感の確認**: 「共感と安堵」が残るよう、負の感情で終わらせずに「小さな喜び」を添える。

## いつ使うか (When to use)
- noteの記事を下書きする際。
- X(Twitter)の投稿案を作成する際。
