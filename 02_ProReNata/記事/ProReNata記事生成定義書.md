## 2. 記事の構成 (Article Structure)
記事は以下の「型」に従って構成されます。

1.  **アイキャッチ**: 1200x630px
2.  **導入文 (Empathy & Structure)**: 
    - 読者の隠している負の感情（泥臭い本音）を言語化し、寄り添う。
    - 「個人のせいではない」ことを論理や比喩で説明し、問題を整理する。
3.  **本文 (Solution)**:
    - **H2見出し**: 3〜4つの大きなテーマで構造化。
    - **H3見出し**: 各テーマ内の詳細。
    - **吹き出し (Speech Bubble)**: セラの実体験や感情（Grounded Empathy）を1割程度織り交ぜる。「セラ（わたし）」が表に出すぎないよう注意。
4.  **まとめ (Summary) [最重要ルール]**:
    - **テキストのみ**: 吹き出し、装飾ブロック、リスト等は使用せず、通常のテキスト（段落）のみで構成。
    - **高い共感と寄り添い**: 「0点」のドライなまとめにならないよう、読者の疲れを労い、前向きな気持ちになれるような**温かい共感の言葉**を必ず含める。
    - **客観的な総括とエンパワーメント**: 記事全体のテーマを総括しつつ、読者が「明日からまたがんばろう」と思えるような、静かな励ましで締めくくる。
    - **アフィリエイト・外部リンク禁止**: Amazon、楽天、ナースリー等のアフィリエイトリンク、および関連記事へのリンクは**一切含めない**（リンクは直後の「あわせて読みたい」セクションに任せる）。
    - **セラ口調の抑制**: 「わたしはずっと応援しています」などの強いキャラ出しは、まとめ内では避ける（地の文としての温かさを優先）。
5.  **あわせて読みたい (Internal Links)**: 関連記事へのリンク。
6.  **免責事項 (Disclaimer)**: 「免責事項」の記述。

## 3. 執筆ルール・表記 (Writing Rules)
基本となる執筆スタイル、「わたし」の一人称、ひらがな化（開き）、世代感の調整、および禁忌事項については、以下のコア・ガイドラインを「唯一の正解」として厳守してください。

- **[03_執筆スタイル (Style_Guidelines).md](file:///Users/sasakiyoshimasa/prorenata/00_システム/UserProfile/03_執筆スタイル(Style_Guidelines).md)**

### ProReNata 特有の補足
- **一行飛ばしの禁止**: 全ての文の後に空行を入れるのは禁止。2〜3文を一つのブロック（段落）としてまとめ、そのブロック間のみ空行を入れます。
- **専門用語の扱い**: 専門用語は、読者（新人看護助手や悩みを持つ方）に合わせて必ず日常的な比喩で補足してください。

## 4. 記事タイプ別テンプレート
1.  **基礎知識・解説系**: 定義・概要 → 重要性/メリット → 判断基準 → まとめ。
2.  **実践ノウハウ系**: 手順 → 現場でのコツ → 避けるべき失敗パターン → まとめ。
3.  **キャリア・転職系**: 現状の課題/選択肢 → 判断基準 → 比較・整理 → まとめ。
