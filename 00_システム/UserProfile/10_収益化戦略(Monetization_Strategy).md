# 10_収益化戦略 (Monetization Strategy)

> **note**: このファイルはCore Contextの一部です。

## 1. コンセプト：導線設計（Flow-First）
「フォロワー数は結果であり、目的ではない」。
少人数のフォロワーであっても、深い信頼と明確な解決策へのルート（導線）があれば、収益化は可能である。
ProReNataにおいては、セラの共感（入口）から、具体的な解決策（出口）へと読者を誘う「線の仕組み」を最優先する。

## 2. 収益化の4ステップ・フロー
1.  **入口（Entrance）：X投稿 / プロフィール**
    - 役割：認知と共感。
    - 指針：単なる嘆きではなく「知見の要約」や「保存したくなるTips」を提供し、プロフィールの解決策（固定ポスト等）へ誘導する。
2.  **信頼（Trust）：note / ブログ記事**
    - 役割：教育とファンの醸成。
    - 指針：セラの実体験を交えた深い洞察を提供し、読者の「知らない人」から「信頼できる案内人」へと関係を深める。
3.  **体験（Experience）：無料コンテンツ・一部公開**
    - 役割：価値の先行提供。
    - 指針：Kindleの無料サンプルや、noteの無料部分で「この先にはもっと深い解決策がある」と確信させる。
4.  **出口（Exit）：収益記事 / サービス / Kindle出版**
    - 役割：具体的な解決とマネタイズ。
    - 指針：読者が「自分への投資」として納得して購入できる、高密度な解決策を提示する。

## 3. 「保存される資産」としての発信
- XでのTips、noteでのまとめ記事、ブログでの比較記事をプロジェクトの「不変資産」として扱う。
- 過去の価値ある投稿をAIが定期的に呼び出し、新しい読者への導線として再利用するサイクルを作る。

## 4. プロフィールの看板化ルール
Xのプロフィールは、セラの「看板」である。
- **Who**: 誰が（精神科病院の看護助手、セラ）
- **What**: 何を解決するか（現場の悩み、人間関係、キャリアの迷い）
- **Where**: どこへ導くか（ブログ、note、Kindle）
この3要素を常に明確に維持し、リンク（導線の終着点）を欠かさない。

## 5. 成功の順番（絶対厳守）
**「まず導線を作り、その後に露出（フォロワー）を増やす」**。
出口のない場所に人を集めない。常に「この記事を読んだ後、読者はどこへ行くべきか？」を自問自答した上でコンテンツを生成する。
